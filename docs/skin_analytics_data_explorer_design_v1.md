## 2. 产品定位

### 2.1 模块定义

Data Explorer 是项目内已发布数据版本的正式浏览与探索页面，支持：

- 查看当前项目/品牌/数据版本下的全量明细数据
- 按品牌、数据集、版本、字段、标签、主键、时间范围进行筛选
- 在表格明细和图表摘要之间快速切换
- 为后续快速分析和高级分析提供变量确认与样本确认入口

### 2.2 目标用户

- 超级管理员
- 租户管理员
- 品牌管理员
- 分析师

### 2.3 使用边界

阶段 2 Data Explorer 的目标是“项目内数据探索”，不是“平台级跨租户明细拼接分析”。

不包含：

- 跨租户明细混合浏览
- 高敏字段明细导出直下
- 高级统计计算取代分析工作台
- 复杂图表编排器取代图表工作台

## 3. 核心能力范围

### 3.1 表格明细浏览

- 分页表格
- 冻结主键列与关键上下文字段
- 显示当前筛选后的记录数、字段数、缺失率摘要
- 支持字段显隐、字段顺序调整、密度切换

### 3.2 筛选与检索

- 品牌筛选
- 数据集筛选
- 数据版本筛选
- 主键检索
- 字段名搜索
- 标签筛选
- 数值范围筛选
- 类别值筛选
- 缺失值筛选

### 3.3 图表查看

阶段 2 最小图表能力：

- 单变量分布图
- 分类构成图
- 双变量散点图
- 箱线图
- 热力图缩略预览

图表主要用于探索，不替代正式分析结果页。

### 3.4 与分析主链路的连接

- 从 Data Explorer 可跳转到快速分析工作台
- 可将当前筛选条件带入分析工作台
- 可将字段选择结果带入高级分析工作台
- 可从结果页回跳到 Data Explorer 查看对应样本明细

## 4. 权限与隔离规则

### 4.1 数据可见范围

- 超级管理员：可见当前租户项目内全部授权品牌数据
- 租户管理员：可见当前租户项目内全部品牌数据
- 品牌管理员：仅可见其授权品牌数据
- 分析师：仅可见其授权品牌数据

### 4.2 字段与导出控制

- 普通浏览默认可显示项目级允许字段
- 高敏字段若后续接入分类管理，默认只显示脱敏态或摘要态
- Data Explorer 不直接提供高敏明细直导
- 如需导出明细，必须进入下载中心/审批链路

### 4.3 品牌切换规则

- 切换品牌后，若当前数据集/版本不属于新品牌，上下文自动重置
- 切换品牌时保留用户最近一次合法筛选草稿
- 若用户拥有多个品牌，允许在同一页面内快速切换，但页面必须重新校验可见字段与数据版本

## 6. 交互要求

### 6.1 明细与图表联动

- 在图表上框选、点击分类项或图例时，可同步更新明细筛选
- 在明细表选中主键集合时，可切换到图表查看对应子样本
- 当前筛选条件必须以可见 token 形式显示，支持逐项清除

### 6.2 性能与体验

- 默认服务器端分页
- 默认只加载当前页明细与当前图表所需摘要
- 复杂图表采用异步刷新，不阻塞表格浏览
- 首屏先加载上下文与记录摘要，再加载明细和图表

### 6.3 容错与提示

- 无数据时显示空状态和可恢复建议
- 筛选过严无结果时显示“放宽条件”建议
- 数据版本失效、冻结或权限变化时提示重新选择版本

## 7. 最小技术方案

### 7.1 后端实现建议

增加独立数据浏览查询服务，职责包括：

- 将当前筛选条件解析为安全查询
- 对项目、品牌、版本、字段权限做统一校验
- 输出明细分页结果
- 输出图表所需聚合摘要
- 为后续图表工作台提供统一查询底座

### 7.2 查询模式

阶段 2 MVP 推荐：

- 基于已发布 `dataset_version`
- 以服务端分页查询明细
- 聚合图表使用预聚合 SQL / pandas 小批量聚合
- 复杂高成本图表延迟加载

### 7.3 返回模型建议

- `ExplorerContext`
- `ExplorerFieldSchema`
- `ExplorerFilterState`
- `ExplorerRowPage`
- `ExplorerChartPreview`

### 7.4 安全约束

- 禁止前端直接拼接 SQL 风格筛选表达式
- 所有字段筛选必须经过后端字段白名单与类型校验
- 主键检索与模糊搜索必须限制长度和频率
- 图表聚合必须继承同一套品牌与字段权限

## 9. 测试与验收建议

### 9.1 P0 测试点

- 超级管理员可查看全部授权品牌数据
- 品牌用户只能查看授权品牌数据
- 品牌切换后上下文自动重校验
- 明细筛选与图表筛选联动正确
- 使用当前筛选条件进入分析工作台成功

### 9.2 验收口径

阶段 2 中，Data Explorer 通过验收的最低标准为：

- 可浏览已发布数据版本的分页明细
- 可按品牌、数据集、版本和字段条件筛选
- 可查看基础探索图表
- 可将当前筛选带入分析主链路
- 浏览器 `core` 级 E2E 通过
