## 3. 全局页面框架

### 顶部栏

包含：

- 当前租户
- 当前项目
- 当前品牌
- 当前用户
- 全局搜索
- 通知中心

### 左侧导航

包含：

- 一级导航
- 最近访问
- 收藏页面

### 内容区

统一原则：

- 顶部显示页面标题与当前上下文
- 次顶部显示关键操作按钮
- 主要内容采用卡片分区

### 全局状态条

在以下场景显示：

- 当前品牌上下文
- 当前数据版本
- 当前 `VersionBundle`
- 当前是否位于平台级受限分析空间

### 4.9.1 全量数据明细浏览页（Data Explorer）

目标：

- 让授权用户在进入统计分析前，先完成数据版本核验、样本筛查、字段浏览和基础图表预览
- 让项目管理者在当前项目上下文中查看多品牌、多数据集、多版本的真实明细数据

适用用户：

- 平台超级管理员
- 租户管理员
- 品牌管理员
- 分析师

权限约束：

- 超级管理员可查看当前项目内全部品牌的数据
- 租户管理员可查看当前租户当前项目内全部授权品牌的数据
- 品牌管理员和分析师仅可查看授权品牌的数据
- 明细导出默认关闭，需跳转到后续审批或导出链路

布局：

- 顶部：项目、品牌、数据集、数据版本上下文带
- 左侧：品牌筛选、数据集筛选、字段树、条件筛选器
- 中部：样本统计卡和明细表
- 右侧：字段摘要卡、分布图、对比图和快捷跳转区

区块：

- 项目上下文区
- 品牌 / 数据集 / 数据版本筛选区
- 字段树与字段搜索区
- 高级条件筛选区
- 样本统计卡
- 明细表
- 图表预览区
- 查询快照与审计提示区

关键组件：

- 品牌多选器
- 数据集选择器
- 数据版本选择器
- 字段树筛选器
- 条件筛选构造器
- 表格分页器
- 固定列切换器
- 图表类型切换器
- 快速分析跳转按钮

支持视图：

- 明细表视图
- 单字段分布视图
- 品牌对比视图
- 样本质量视图

主流程：

1. 选择品牌范围
2. 选择数据集与数据版本
3. 通过字段树和高级筛选器缩小样本范围
4. 查看样本统计与明细记录
5. 选择字段生成基础图表预览
6. 将当前筛选结果带入快速分析或高级分析

关键动作：

- 切换品牌 / 数据集 / 数据版本
- 保存查询条件
- 重置筛选
- 查看单条样本详情
- 排序、分页、固定列
- 预览图表
- 进入快速分析
- 进入高级分析

### 4.10 快速分析工作台

目标：

- 给普通用户快速完成标准分析

布局：

- 左侧：品牌、数据版本、分析目的、目标变量
- 中部：系统推荐方法卡
- 右侧：结果预览摘要

关键组件：

- 变量投放器
- 方法推荐卡
- 结果摘要卡

主流程：

1. 选择品牌
2. 选择数据版本
3. 选择分析目标
4. 系统推荐方法
5. 运行分析

页面特征：

- 独立入口

- 明确标识“平台级受限分析空间”

区块：

- 多租户、多品牌数据选择区
- 聚合分析参数区
- 高敏导出申请区
- 强审计提示区

### 4.18 问卷模块与主平台连接点

- 项目初始化页绑定问卷模板版本

- 问卷结果管理页可将总分/维度分发布为派生变量
- 分析工作台可直接选择问卷派生变量
- 全量数据明细浏览页可将当前筛选结果带入分析工作台

### 4.19 平台管理页

区块：

- 租户管理
- 系统默认配置
- 受限分析空间入口
- 审计总览

## 5. 页面跳转关系

### 主链路

1. 登录页
2. 租户工作台
3. 项目详情
4. 上传向导或分析工作台
5. 结果详情
6. 图表工作台 / 下载中心

### 数据链路

1. 数据集列表
2. 上传向导
3. 数据预处理
4. 版本发布
5. 全量数据明细浏览
6. 分析工作台

### 问卷链路

1. 问卷模板页
2. 问卷发放页
3. 问卷预览与答卷页
4. 问卷历史导入页
5. 问卷结果管理页
6. 计分结果
7. 派生变量发布

### 分析链路

1. 品牌选择
2. 数据版本选择
3. 快速分析 / 高级分析
4. 异步任务
5. 结果详情
6. 派生变量发布

### 高敏导出链路

1. 结果详情
2. 发起导出
3. 待审批
4. 审批通过
5. 下载中心

## 6. 品牌切换与状态保留

切换品牌时保留：

- 项目上下文
- 用户偏好
- 已收藏页面

切换品牌时重校验：

- 变量投放
- 分组规则
- 分析模板
- 图表模板

切换品牌时清空：

- 当前品牌不可用变量
- 当前品牌不可用模板实例

## 7. 关键交互说明

### 7.1 快速分析

- 适合业务用户
- 默认隐藏复杂参数
- 输出主图 + 主表 + 简版解释

### 7.2 高级分析

- 适合分析师
- 暴露完整参数
- 支持手动覆盖系统推荐

### 7.3 派生变量发布

- 在结果详情页完成
- 选择变量后发布为新版本
- 跳转新版本详情页

### 7.4 高敏导出

- 导出前判断是否命中高敏策略
- 命中则进入审批单流程
- 审批通过后下载
