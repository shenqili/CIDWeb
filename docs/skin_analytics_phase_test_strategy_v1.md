## 2. 阶段 0 测试

- 单元：无
- 集成：环境脚本与配置校验
- API：`/health`
- E2E：首页可访问、健康可访问
- 证据：容器状态、首页截图、smoke 报告

## 3. 阶段 1 测试

- 单元：字段映射、主键映射、版本状态流
- 集成：上传入库、批次写入、品牌绑定
- API：项目、品牌、`VersionBundle`、数据集、上传
- 权限/视图：品牌用户可见性、品牌范围控制
- E2E：项目初始化 -> 品牌用户授权 -> 品牌指标配置确认 -> 上传 -> 发布版本
- 证据：权限截图、上传截图、主键冲突截图、版本发布截图、API 日志

## 4. 阶段 2 测试

- 单元：方法推荐逻辑、结果摘要组装
- 集成：分析任务与任务中心
- API：analysis / tasks / derived publish
- 权限/视图：品牌切换、模板校验、未授权品牌访问拦截、上下文重校验
- E2E：快速分析、高级分析、结果详情、派生变量发布
- 可视化：主图与主表一致性

## 5. 阶段 3 测试

- 单元：问卷逻辑、计分、跳题
- 集成：问卷答卷入库、审批流
- API：questionnaires / approval
- 权限/视图：匿名问卷、项目级规则改写
- E2E：匿名填写、实时答卷、历史导入、高敏导出审批
- 证据：问卷截图、异常清单、审批状态截图

## 6. 阶段 4 测试

- 单元：统计方法包装器
- 集成：高级方法执行与导出
- API：analysis advanced
- 统计方法：GLM / 因子 / 聚类 / 问卷列联
- 可视化：森林图、载荷热图、残差热图
- E2E：高级分析闭环
- 证据：黄金数据集比对、图表截图、执行快照

## 7. 阶段 5 测试

- 单元：权限与审批规则
- 集成：受限空间与审计
- API：restricted-analytics / export approvals
- 权限/视图：超级管理员、普通用户越权、跨品牌混合分析限制
- E2E：受限空间聚合分析、明细导出审批
- 证据：警示页截图、审计日志、审批状态

## 8. 阶段 6 测试

- Smoke：环境与最短主链
- Core：各阶段核心链路
- Full：异常链路与高级能力，覆盖空主键、非法字段映射、重复发布幂等、XLSX 浏览器上传专项
- 演示验收：账号、数据、脚本、报告
- 证据：HTML 报告、截图、视频、日志归档
