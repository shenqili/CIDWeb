# 企业级皮肤大数据管理与智能分析平台 页面线框级说明 v1

## 1. 目标

本文件在页面原型说明基础上，将关键页面进一步压缩为低保真线框结构，供 UI、前端、测试统一理解页面层级与状态流。

---

## 2. 全局骨架

### 标准工作台骨架

- 顶部栏
- 左侧导航
- 页面标题区
- 操作栏
- 主内容区

### 平台级受限分析空间骨架

- 顶部警示条
- 平台级上下文条
- 左侧简化导航
- 主分析区域

---

## 3. 关键页面线框

### 登录页

- 顶部平台标识
- 中央登录卡
- 底部安全与隐私提示

### 工作台

- 第一行：项目卡片、待审批卡片、最近任务卡片
- 第二行：最近上传、最近分析、风险提醒

### 项目详情页

- 顶部：项目名、状态、操作按钮
- 左：项目信息、成员列表
- 右：品牌绑定、VersionBundle、导出策略

### 品牌配置页

- 左：品牌列表
- 中：配置版本列表
- 右：版本详情与发布按钮

### 数据集列表页

- 顶部筛选条
- 中央数据集表
- 右上快速上传入口

### 数据预处理页

- 顶部版本状态条
- 左：字段概览
- 中：缺失值与异常值处理区
- 右：样本预览与发布按钮

### 全量数据浏览页（Data Explorer）

- 顶部：项目、品牌、数据集、数据版本上下文带
- 左：品牌/数据集/版本筛选器、字段搜索、字段标签筛选、样本筛选构造器
- 中：明细表格、排序、分页、显示列控制
- 右：数据概览卡、缺失值摘要、字段分布图、品牌或版本对比图
- 底部：保存视图、推送到分析工作台、申请明细导出

### 上传向导页

- 顶部步骤条
- 左：文件信息和上下文
- 中：步骤主操作区
- 底部：上一步 / 下一步 / 保存草稿

### 快速分析工作台

- 左：品牌、数据版本、分析目的、目标变量
- 中：系统推荐方法卡
- 右：结果摘要占位

### 高级分析工作台

- 左：字段树、搜索、收藏、标签筛选
- 中：变量投放器、参数区、方法说明
- 右：摘要卡、图表预览、诊断标签页

### 结果详情页

- 顶部：品牌、数据版本、VersionBundle、方法
- 标签：摘要 / 明细 / 图表 / 方法 / 日志
- 图表区：主图 + 二级图形切换器

### 图表工作台

- 左：图种列表
- 中：大图预览
- 右：属性配置

### 下载中心

- 顶部筛选条
- 中央导出包列表
- 右侧审批状态与有效期信息

### 任务中心

- 顶部筛选条
- 中央任务表
- 右侧失败日志抽屉

### 审计日志页

- 顶部高级筛选区
- 中央日志表
- 右侧高风险事件卡

### 问卷模板页

- 左：模板列表
- 中：题目结构树
- 右：题目编辑、逻辑、计分

### 问卷预览与答卷页

- 顶部：问卷标题、进度、上下文
- 中央：题目渲染区
- 底部：上一页 / 下一页 / 提交

适用模式：

- 项目内登录填写
- 外部匿名链接填写
- 二维码移动端填写

### 问卷发放页

- 发放方式卡
- 链接区
- 二维码区
- 有效期配置
- 回收统计

### 问卷历史导入页

- 上传区
- 模板匹配区
- 编号映射区
- 异常提示区

### 平台级受限分析空间

- 顶部红/橙色警示条
- 多租户、多品牌选择器
- 聚合分析参数面板
- 聚合结果区
- 高敏导出申请区

### 平台管理页

- 左：租户列表
- 中：系统默认配置
- 右：平台级审计与受限分析入口

## 4. 页面状态

### 品牌切换

- 切换前确认弹窗
- 兼容性校验结果
- 不兼容对象清空提示

### 任务状态

- `queued`
- `running`
- `partial_success`
- `failed`
- `succeeded`

### 审批状态

- `pending_approval`
- `approved`
- `rejected`
- `expired`

### 问卷状态

- `draft`
- `submitted`
- `invalid`
- `scored`
- `pending_match`
