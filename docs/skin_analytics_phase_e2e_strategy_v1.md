## 2. 统一规则

### 2.1 E2E 分层

- `smoke`
  只验证环境和最短主链路是否可用
- `core`
  验证当前阶段核心业务链路
- `full`
  验证当前阶段全部业务和异常链路

### 2.2 统一证据留存

每次阶段 E2E 执行至少保留：

- 执行时间
- 环境版本
- 通过/失败清单
- 失败截图
- 失败视频
- 关键接口日志
- Docker 容器状态

### 2.3 Playwright 对应关系

- `smoke`：直接对应当前 `tests/e2e/smoke`
- `core`：对应当前阶段新增的业务 spec
- `full`：对应当前阶段全部 spec + 关键异常 spec

## 3. 各阶段 E2E 规划

### 阶段 0：项目基线与脚手架

#### 用例范围

- 首页可访问
- 健康接口可访问
- Docker 环境启动成功

#### 进入条件

- `docker-compose.demo.yml` 已可用
- 前后端最小壳已存在
- `Playwright` 运行环境已装好

#### 退出条件

- `smoke` 全通过
- 首页 `200`
- 健康接口 `200`

#### 证据留存

- `docker compose ps`
- smoke 测试结果
- 首页截图

### 阶段 1：数据底座

#### 用例范围

- 项目初始化
- 多品牌绑定
- 品牌用户授权
- 品牌指标配置确认
- 品牌配置版本发布
- 上传标准文件
- 字段映射
- 主键冲突处理
- 数据版本发布

#### 进入条件

- 阶段 0 E2E 已通过
- 后端数据接入 API 已可访问
- 前端上传页面已可操作

#### 退出条件

- `smoke + core` 通过
- 能完成“项目初始化 -> 品牌用户授权 -> 品牌指标配置确认 -> 上传 -> 发布版本”
- 上传入库的数据与品牌、`VersionBundle` 绑定关系正确

#### 证据留存

- 品牌用户权限截图
- 品牌指标配置发布截图
- 上传结果截图
- 主键映射冲突截图
- 数据版本发布结果截图
- API 日志

### 阶段 2：分析主链路

#### 用例范围

- 快速分析
- 高级分析基础链路
- 品牌切换
- 结果详情页
- 派生变量发布
- 任务中心状态流

#### 进入条件

- 阶段 1 核心用例通过
- 分析工作台、结果页、任务中心可访问

#### 退出条件

- `core` 通过
- 可完成“发布版本 -> 分析 -> 查看结果 -> 发布派生变量”

#### 证据留存

- 快速分析页面截图
- 高级分析页面截图
- 结果页截图
- 派生变量发布后新版本截图

### 阶段 3：问卷模块与审批链路

#### 用例范围

- 问卷模板发布
- 外部匿名填写
- 在线实时答卷
- 历史问卷导入
- 问卷计分
- 高敏导出审批通过/拒绝/过期

#### 进入条件

- 阶段 2 核心用例通过
- 问卷模板、问卷发放、问卷结果、高敏导出页面可访问

#### 退出条件

- `core + 问卷专项 + 审批专项` 通过
- 匿名填写、历史导入和审批链路都闭环

#### 证据留存

- 外部匿名问卷填写截图
- 历史导入异常清单截图
- 审批状态截图
- 下载前后状态截图

### 阶段 4：高级分析能力

#### 用例范围

- GLM
- 因子分析
- 聚类分析
- 问卷列联分析增强
- 高级图表导出

#### 进入条件

- 阶段 3 核心用例通过
- 高级方法 UI 与 API 已可用

#### 退出条件

- `full` 中高级分析相关用例通过
- 黄金数据集比对达标

#### 证据留存

- 主统计量比对结果
- 森林图、载荷热图、残差热图导出截图
- 执行快照摘要

### 阶段 5：平台治理与受限空间

#### 用例范围

- 超级管理员进入受限分析空间
- 跨品牌、跨租户聚合分析
- 默认仅聚合输出
- 明细导出申请与审批
- 清理与保留策略验证

#### 进入条件

- 阶段 4 通过
- 超级管理员专属入口可访问

#### 退出条件

- 受限空间 `core/full` 用例通过
- 普通用户越权访问被拦截

#### 证据留存

- 受限空间警示页截图
- 聚合结果页截图
- 明细导出审批状态截图
- 审计日志截图

### 阶段 6：自动化与演示收口

#### 用例范围

- Smoke 套件
- Core 套件
- Full 套件
- 演示数据与演示账号验证
- 演示脚本走查

#### 进入条件

- 前五阶段核心功能闭环

#### 退出条件

- Smoke 稳定通过
- Core 可用于回归
- Full 可用于阶段性全量验收

#### 证据留存

- HTML 测试报告
- 失败截图/录像
- 演示脚本记录
